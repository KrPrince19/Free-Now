## üß† Smart Logic & Features

### 1. Chat Persistence (The Refresh Fix)
- **Mechanism**: The app uses `localStorage` to remember if you were in a chat.
- **How it works**: If you refresh the page while chatting, the code checks "Did I have a chat open?" before the page loads. If yes, it instantly re-renders the ChatBox and re-connects the socket, so your conversation never breaks.

### 2. Real-time "Talking Now" Pulse
- **Mechanism**: Global Broadcasts.
- **How it works**: Whenever two people connect anywhere in the world, the server sends a "Vibe Started" signal. The landing page hears this and adds them to the "Talking Now" list instantly. It uses a "Pulse" animation to show the connection is live.

### 3. Smart Theme Sync
- **Mechanism**: Context API.
- **How it works**: The app has a **ThemeContext** that wraps the entire project. When you toggle Dark Mode on the Profile page, the Landing page and Dashboard automatically "hear" this change and update their colors, background hearts, and shadows to match.

### 4. Advanced Chat Actions
- **Editing**: Uses Socket.io (`edit-message` / `message-updated`) for real-time synchronization. The frontend switches to "Edit Mode," populates the input, and broadcasts changes globally, adding an lowercase "‚Ä¢ edited" badge for transparency.
- **Deletion**: Implements a "placeholder" pattern via socket events. Instead of a hard delete, the message content is replaced with "This message was deleted" and styled with italics/gray colors, preserving the conversation flow for both users.
- **Copying**: Built-in clipboard integration (`navigator.clipboard`) with a 2-second visual confirmation state (swapping the copy icon for a green checkmark) for a smooth user experience.

### 5. Typewriter Effect
- **Mechanism**: Timed String Manipulation.
- **How it works**: In the Hero section, a custom script cycles through romantic and "lovable" sentences, typing them out character by character to create a welcoming, poetic vibe.
