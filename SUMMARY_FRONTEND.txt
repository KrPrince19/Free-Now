## üß† Smart Logic & Features

### 1. Chat Persistence (The Refresh Fix)
- **Mechanism**: The app uses `localStorage` to remember if you were in a chat.
- **How it works**: If you refresh the page while chatting, the code checks "Did I have a chat open?" before the page loads. If yes, it instantly re-renders the ChatBox and re-connects the socket, so your conversation never breaks.

### 2. Real-time "Talking Now" Pulse
- **Mechanism**: Global Broadcasts.
- **How it works**: Whenever two people connect anywhere in the world, the server sends a "Vibe Started" signal. The landing page hears this and adds them to the "Talking Now" list instantly. It uses a "Pulse" animation to show the connection is live.

### 3. Smart Theme Sync
- **Mechanism**: Context API.
- **How it works**: The app has a **ThemeContext** that wraps the entire project. When you toggle Dark Mode on the Profile page, the Landing page and Dashboard automatically "hear" this change and update their colors, background hearts, and shadows to match.

### 5. Snapshot Ephemeral Media (True Privacy)
- **Mechanism**: Auto-Destructing Payload.
- **How it works**: In the ChatBox, users can send images as "Snapshots". These images are only revealed when the recipient clicks them. Once opened, a **10-second timer** starts. When it hits zero, the image is permanently wiped from the chat state, ensuring sensitive media never sticks around.

### 6. Vibe-Based Discovery
- **Mechanism**: Real-time Client Filtering.
- **How it works**: The Dashboard now includes a "Vibe Bar". Users can instantly filter the list of online people by their vibe category (Chill, Deep, Chaotic, etc.), making it easier to find meaningful connections.

### 7. High-Performance Sonar Map (Visibility Fix)
- **Mechanism**: Multi-Source State Merging.
- **How it works**: The previous multi-theme globe map was simplified into a single, high-performance **Sonar Radar**. Crucially, it was fixed to show **EVERYONE** online (both individual "Free" users and matched pairs), providing a 100% accurate visual representation of the community's real-time energy.

### 6. Dynamic HSL Backgrounds (Global Vibe Pulse)
- **Mechanism**: Global CSS Variable (`--vibe-hue`) driven by `ThemeContext`.
- **How it works**: The app automatically monitors the number of active users. When the network is quiet, the backgrounds shift to a calm **Indigo** hue (240¬∞). As more people connect and the "vibe" increases, the entire app's background accents smoothly transition towards a vibrant **Rose** hue (340¬∞), reflecting the energy of the community in real-time.
